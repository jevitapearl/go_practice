# go_practice

These are a series of files in which I have practiced coding with Golang.

All the Go lang fundamentals were covered from a course by Tech with Tim on YouTube
I've covered the topics till error handling. I'll further be pushing code as I learn concepts over the time.
Link: https://youtu.be/V-lI7AmusGs?si=OZoYWX8NdkRvBAJ0

Usage:

1. Clone the repo
2. Install Go lang
3. Go to main.go file and uncomment the file you want to execute
4. Go to terminal and run "go run ."
